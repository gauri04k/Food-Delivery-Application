# Food Delivery Application 🍔

A console-based Food Delivery Management System developed using Java, Eclipse IDE, and the Collections Framework. This project follows Object-Oriented Programming principles and uses POJO (Plain Old Java Object) classes to represent core entities such as Customer, Product, and Order.

---

## 📌 Features

* Product Management
* Customer Management
* Order Processing
* Revenue Calculation
* Product Filtering by Price
* Product Search by Name
* Customer Search by City
* Order Search by Customer
* Product Sorting using Comparator
* Data Storage using List and HashMap

---

## 🛠 Technologies Used

* Java
* Eclipse IDE
* Collections Framework
* Object-Oriented Programming (OOP)
* POJO Classes
* Comparator
* HashMap
* ArrayList

---

## 🏗 POJO Classes

### Customer.java

Represents customer information including:

* Customer ID
* Customer Name
* Customer City

### Product.java

Represents food items including:

* Product ID
* Product Name
* Product Price

### Order.java

Represents order details including:

* Order ID
* Customer Information
* Ordered Products
* Total Amount
